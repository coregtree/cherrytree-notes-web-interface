# cherrytree-notes-web-interface
A web interface upgrade for cherrytree notes providing a significant upgrade to the default website interface given by cherytree when using the "Export as HTML" feature. This interface remains fully client side. It provides full instant (through cache files) image and text search across all pages, makes the interface mobile friendly, adds a dark theme, and adds a font size resizing feature. Now any device can become a cherrytree notes viewer if you host your notes online as a static site.
